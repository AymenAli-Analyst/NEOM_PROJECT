# NEOM Saudi Vision 2030 Data Analysis 📊

## 🌟 Overview
Strategic data analysis of Saudi market sectors, providing deep insights into company performances and revenue trends. This project demonstrates the full data cycle from cleaning to visualization.

## 🖼️ Dashboard Preview
![NEOM Dashboard](NEOM_DASH.png)

## 🛠️ Tech Stack
* **SQL:** Data cleaning, processing, and complex querying of over 1M records.
* **Power BI:** Designing interactive dashboards and visual storytelling.
* **Excel:** Initial data exploration and trend analysis.

## 📈 Key Insights & Results
* **Top Profitability:** ACWA Power achieved the highest profit at **7.3K**.
* **Revenue Growth:** Massive revenue spike in 2024, reaching **38.5K**.
* **Market Stability:** Maintained a consistent average profit margin of **17.32%**.

## 📁 Repository Contents
* `SQLQuery2.sql`: The SQL scripts used for data cleaning.
* `NEOM_Project.pbix`: The interactive Power BI dashboard file.
* `NEOM_Saudi_Vision2030_Project_Data.xlsx`: The raw dataset used for analysis.
